*   **Teks untuk Canva**:
    *   *Country*: `INDONESIA 🇮🇩`
    *   *Project Title*: `INTELLIGENT RETAIL INVENTORY DECISION SUPPORT SYSTEM FOR DEADSTOCK OPTIMIZATION`
    *   *Team Info*: `Team Deadstock Solvers • Universitas [Nama Kampus Anda]`

---

### 2. 📘 BOX 1: INTRODUCTION (Kolom Kiri - Atas)
*   **Panduan Visual di Canva**:
    *   Buat kotak (*card*) berlatar belakang putih dengan garis tepi hitam/gelap dan ujung membulat.
    *   Tulis judul **"Introduction"** di pojok kiri atas kotak dengan huruf tebal.
    *   Gunakan sub-judul berwarna merah untuk bagian *Problem* dan berwarna hijau untuk bagian *Solution*.
*   **Teks untuk Canva**:
    *   **Problem (Capital Stagnation & Inefficient Supply Chain)**:
        *   Retail businesses frequently struggle with cash flow blockages caused by accumulated unsold items (Deadstock).
        *   Store managers lack automated tools to analyze inventory velocity, leaving procurement decisions to manual intuition, which leads to stockouts or overstocking.
    *   **Solution (Dynamic Analytics & Decision Support)**:
        *   Developed a collaborative web-based Decision Support System (DSS) using a 3-month rolling average to dynamically categorize products.
        *   Integrated transaction-safe Excel/CSV sales import with strict database rollbacks to guarantee data integrity.

---

### 3. ⚙️ BOX 2: METHODE / METHODOLOGY (Kolom Kanan - Atas)
*   **Panduan Visual di Canva**:
    *   Buat kotak (*card*) berlatar belakang putih dengan ujung membulat.
    *   Tulis judul **"Methode"** di pojok kiri atas kotak.
    *   **Elemen Visual**: Buat bagan alir (*flowchart*) sederhana horizontal menggunakan bentuk kotak dan panah di Canva:
        `[1. Core Design] ──> [2. DSS Algorithm] ──> [3. Bulk Importer] ──> [4. Testing]`
    *   Di bawah flowchart, cantumkan penjelasan poin-poinnya.
*   **Teks untuk Canva**:
    1.  **Architecture Design**: Relational database schema linking users, products, categories, suppliers, and stock movements.
    2.  **DSS Algorithm**: Programmed mathematical logic evaluating 3-month rolling sales.
    3.  **Integrity Controls**: Built database transactions on Excel/CSV imports to roll back and abort operations upon encountering invalid rows.
    4.  **Visualization & Testing**: Real-time sales charts rendering and testing with 2 years of simulated retail data.

---

### 4. 📊 BOX 3: DATA AND DISCUSSION (Kolom Kiri - Bawah)
*   **Panduan Visual di Canva**:
    *   Buat kotak (*card*) membulat yang sedikit lebih panjang ke bawah.
    *   Tulis judul **"Data and Discussion"** di pojok kiri atas kotak.
    *   **Elemen Visual 1 (Rumus & Tabel)**: Ketik rumus rata-rata bulanan dan buat tabel 3 baris berisi klasifikasi status inventaris (Deadstock, Fast-Moving, Slow-Moving). Berikan warna latar belakang kontras pada teks status (Merah untuk Deadstock, Hijau untuk Fast, Kuning untuk Slow).
    *   **Elemen Visual 2 (Grafik Batang)**: Buat grafik batang sederhana di Canva untuk menunjukkan sebaran stok (Misal: Batang Merah tinggi untuk Deadstock (300 pcs), Batang Hijau pendek untuk Fast-Moving (20 pcs), Batang Kuning sedang untuk Slow-Moving (150 pcs)).
*   **Teks untuk Canva**:
    *   *Formula*:
        $$A_{monthly} = \text{round}\left(\frac{\text{Total Sales in Last 3 Months}}{3}\right)$$
    *   *Tabel Aturan Klasifikasi*:
        *   **Deadstock**: Sales (3M) = 0 | Recommendation: *Jangan Beli (Obral)* | Suggested Buy = 0
        *   **Fast-Moving**: Sales (3M) > 0 & Stock < 1.5 * A_monthly | Recommendation: *Beli Banyak (Urgent)* | Suggested Buy = (A_monthly + Safety Stock) - Current Stock
        *   **Slow-Moving**: Sales (3M) > 0 & Stock >= 1.5 * A_monthly | Recommendation: *Beli Sedikit / Tunda* | Suggested Buy = (A_monthly + Safety Stock) - Current Stock

---

### 5. 🏆 BOX 4: CONCLUSION (Kolom Kanan - Bawah)
*   **Panduan Visual di Canva**:
    *   Buat kotak (*card*) membulat.
    *   Tulis judul **"Conclusion"** di pojok kiri atas kotak.
    *   Gunakan ikon/ilustrasi kecil di Canva di samping setiap poin manfaat (misal: ikon koin/uang untuk finansial, ikon petir untuk efisiensi, dan ikon gembok untuk keamanan data).
*   **Teks untuk Canva**:
    *   **Prevents Stagnant Capital**: Identifies obsolete/deadstock items instantly, allowing managers to run clearance sales to free up stagnant working capital.
    *   **Optimizes Inventory Turn**: Suggests precise reorder quantities to avoid stockouts on high-demand items while preventing excess inventory.
    *   **Guarantees Data Quality**: Ensures zero stock discrepancy through atomic bulk spreadsheet imports.
